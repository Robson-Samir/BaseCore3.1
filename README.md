# Arquitetura base de microservice 

Desenvolvido na linguagem C# e utiliza a versão .Netcore 3.1

## Padrões implantados
- [x] Designer partners como Dry (evitar ambiguidade), Kiss (funções simples), Solid (l - Segregação de interface, S - Resposabilidade única, D - Inversão de dependência) 
- [x] DDD
- [x] Arquitetura de MicroServiços
- [x] Camada para Jobs(Tarefas automatizadas)
- [x] Camada para testes
- [x] Injeção de dependência
- [x] Multi Banco de dados

## Ambiente
Pacote disponível para todos os tipos de servidores
